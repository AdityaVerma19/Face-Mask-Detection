# 🎭 Real-Time Face Mask Detection System

A real-time face mask detection system built using **MobileNetV2 (Transfer Learning)** and **OpenCV DNN face detector**.

## Features
- Transfer Learning using MobileNetV2
- Fine-Tuning of last 20 layers
- EarlyStopping for optimal training
- Confusion Matrix & Performance Evaluation
- Real-time webcam detection
- ~99% validation accuracy

## Tech Stack
- Python 3.10
- TensorFlow / Keras
- OpenCV
- NumPy

## Model Performance
- Accuracy: ~99%
- Precision: 0.99
- Recall: 0.99
- F1-Score: 0.99

## How to Run

1. Clone repository
2. Create virtual environment
3. Install dependencies: pip install -r requiremnts.txt     python
4. Run:detect_mask.py
Press 'q' to quit.
